# 🏛️ ECOMMERCE-API MASTERPLAN

**AUTHOR:** Aditya Maisuriya
**PURPOSE:** Backend Architecture and Database Schema for the Ecommerce Platform.

## 🌍 OVERVIEW
A robust, highly scalable .NET Core Web API serving the backend needs of the Ecommerce system.

## 🏗️ ARCHITECTURE
- **Framework:** ASP.NET Core
- **Database:** Entity Framework Core (SQL Server / PostgreSQL)
- **Patterns:** Repository Pattern, DTOs, Middleware for JWT Auth.

## 🚀 DEPLOYMENT
- CI/CD via GitHub Actions.
- Docker containerization ready.
